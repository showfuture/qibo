IP设置
允许进入整站后台的IP:
(留空则不限制,你可以设置只允许哪些IP进入后台,每个IP换一行,比如
192.168.0.2
192.168.1.5
192.168.0

如果某行设置成192.168.0 那么192.168.0.1-192.168.0.255这个范围内的IP都是在允许的范围内
你当前的IP是: $onlineip
注意:如果你的IP不是固定的话,比如是ADSL拨号的话,请不要填写,否则IP更换后就进不了后台,此时只能通过修改后台文件强制进入后台进行修改恢复
)
允许访问前台网站的IP:
(留空则不限制,你可以设置只允许哪些IP访问网站,每个IP换一行,比如
192.168.0.2
192.168.1.5
192.168.0

如果某行设置成192.168.0 那么192.168.0.1-192.168.0.255这个范围内的IP都是在允许的范围内
注意:这里只限制访问前台,但不限制登录后台
)
封IP:
(你可以设置封掉哪些IP访问网站前台和后台,每个IP换一行,比如
192.168.0.2
192.168.1.5
192.168.0

如果某行设置成192.168.0 那么192.168.0.1-192.168.0.255这个范围内的IP都是在受封的范围内
注意:这里会同时禁止访问前台与后台,请不要封自己的IP,否则很麻烦,你需要修改文件data/config.php查找到你的IP把他删除才行
)
